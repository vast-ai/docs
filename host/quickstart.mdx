## Purpose

Give a first-time host one ordered path instead of a long omnibus page.

## Proposed Journey

1. Decide whether Vast fits your hardware and expectations.
2. Create or convert a host account.
3. Prepare Ubuntu, drivers, BIOS, storage, and networking.
4. Install the Vast host software.
5. List the machine.
6. Run self-test.
7. Understand verification and first-rental expectations.

Each step should link to a deeper page.
