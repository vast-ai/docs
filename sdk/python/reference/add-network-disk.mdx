Add a network disk to a physical cluster.

## Signature

```python
VastAI.add_network_disk(
    machines: List[int],
    mount_point: str,
    disk_id: Optional[int] = None
) -> dict
```

## Parameters

<ParamField path="machines" type="List[int]" required>
  ids of machines to add disk to, that is networked to be on the same LAN as machine
</ParamField>

<ParamField path="mount_point" type="str" required>
  mount path of disk to add
</ParamField>

<ParamField path="disk_id" type="Optional[int]">
  id of network disk to attach to machines in the cluster
</ParamField>

## Returns

`dict`

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.add_network_disk(machines=12345, mount_point="value")
print(result)
```
