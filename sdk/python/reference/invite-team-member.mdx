Invite a new member to the team.

## Signature

```python
VastAI.invite_team_member(email: Optional[str] = None, role: Optional[str] = None) -> str
```

## Parameters

<ParamField path="email" type="Optional[str]">
  Email address of the person to invite.
</ParamField>

<ParamField path="role" type="Optional[str]">
  Role to assign to the new team member.
</ParamField>

## Returns

`str` — Result from the API call.

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.invite_team_member()
print(result)
```
