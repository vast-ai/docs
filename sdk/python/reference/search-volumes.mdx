Search for volume offers using custom query.

## Signature

```python
VastAI.search_volumes(query: Optional[str] = None, storage: float = 1.0, order: str = "score-") -> str
```

## Parameters

<ParamField path="query" type="Optional[str]">
  Query to search for. default: 'external=false verified=true disk_space&gt;=1', pass -n to ignore default
</ParamField>

<ParamField path="storage" type="float" default="1.0">
  Amount of storage to use for pricing, in GiB. default=1.0GiB
</ParamField>

<ParamField path="order" type="str" default="&quot;score-&quot;">
  Comma-separated list of fields to sort on. postfix field with - to sort desc. ex: -o 'disk_space,inet_up-'.  default='score-'
</ParamField>

## Returns

`str` — Result from the API call.

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.search_volumes()
print(result)
```
