Activate a new 2FA method by verifying the code.

## Signature

```python
VastAI.tfa_activate(
    code: str,
    sms: bool = False,
    secret: Optional[str] = None,
    phone_number: Optional[str] = None,
    label: Optional[str] = None
) -> str
```

## Parameters

<ParamField path="code" type="str" required>
  6-digit verification code from SMS or Authenticator app
</ParamField>

<ParamField path="sms" type="bool" default="False">
  sms
</ParamField>

<ParamField path="secret" type="Optional[str]">
  Secret token from setup process (required)
</ParamField>

<ParamField path="phone_number" type="Optional[str]">
  Phone number for SMS method (E.164 format)
</ParamField>

<ParamField path="label" type="Optional[str]">
  Label for the new 2FA method
</ParamField>

## Returns

`str` — Result from the API call.

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.tfa_activate(code="...")
print(result)
```
