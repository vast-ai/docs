Delete a template by ID or hash.

## Signature

```python
VastAI.delete_template(
    template_id: Optional[int] = None,
    hash_id: Optional[str] = None
) -> dict
```

## Parameters

<ParamField path="template_id" type="Optional[int]">
  Template ID of Template to Delete
</ParamField>

<ParamField path="hash_id" type="Optional[str]">
  Hash ID of Template to Delete
</ParamField>

## Returns

`dict`

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.delete_template()
print(result)
```
