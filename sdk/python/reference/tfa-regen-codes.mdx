Regenerate backup codes for 2FA. Invalidates all previously generated backup codes.

## Signature

```python
VastAI.tfa_regen_codes(
    code: Optional[str] = None,
    sms: bool = False,
    secret: Optional[str] = None,
    backup_code: Optional[str] = None,
    method_id: Optional[str] = None
) -> str
```

## Parameters

<ParamField path="code" type="Optional[str]">
  The 2FA code to authorize this action.
</ParamField>

<ParamField path="sms" type="bool" default="False">
  Use SMS delivery for the verification code.
</ParamField>

<ParamField path="secret" type="Optional[str]">
  Secret token from the previous login step (required for SMS or Email 2FA).
</ParamField>

<ParamField path="backup_code" type="Optional[str]">
  One-time backup code (alternative to regular 2FA code).
</ParamField>

<ParamField path="method_id" type="Optional[str]">
  2FA method ID when you have more than one method of the same type.
</ParamField>

## Returns

`str` — Result from the API call.

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.tfa_regen_codes()
print(result)
```
