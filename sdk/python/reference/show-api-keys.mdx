Return all API keys associated with the account.

## Signature

```python
VastAI.show_api_keys() -> list[dict]
```

## Returns

`list[dict]`

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.show_api_keys()
print(result)
```
