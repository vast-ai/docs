Create a new network volume.

## Signature

```python
VastAI.create_network_volume(
    id: int,
    size: float = 15,
    name: Optional[str] = None
) -> dict
```

## Parameters

<ParamField path="id" type="int" required>
  id of network volume offer
</ParamField>

<ParamField path="size" type="float" default="15">
  size in GB of network volume. Default %(default)s GB.
</ParamField>

<ParamField path="name" type="Optional[str]">
  Optional name of network volume.
</ParamField>

## Returns

`dict`

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.create_network_volume(id=12345)
print(result)
```
