Update a 2FA method's settings.

## Signature

```python
VastAI.tfa_update(method_id: int, label: Optional[str] = None, set_primary: Optional[str] = None) -> str
```

## Parameters

<ParamField path="method_id" type="int" required>
  ID of the 2FA method to update (see `vastai tfa status`)
</ParamField>

<ParamField path="label" type="Optional[str]">
  New label/name for this 2FA method
</ParamField>

<ParamField path="set_primary" type="Optional[str]">
  Set this method as the primary/default 2FA method
</ParamField>

## Returns

`str` — Result from the API call.

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.tfa_update(method_id=12345)
print(result)
```
