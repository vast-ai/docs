Update an existing team role.

## Signature

```python
VastAI.update_team_role(
    id: int,
    name: Optional[str] = None,
    permissions: Optional[str] = None
) -> dict
```

## Parameters

<ParamField path="id" type="int" required>
  id of the role
</ParamField>

<ParamField path="name" type="Optional[str]">
  name of the role
</ParamField>

<ParamField path="permissions" type="Optional[str]">
  file path for json encoded permissions, look in the docs for more information
</ParamField>

## Returns

`dict`

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.update_team_role(id=12345)
print(result)
```
