Show details of a hosted machine.

<Note>This is a **host** method, used for managing machines you are renting out on Vast.ai.</Note>

## Signature

```python
VastAI.show_machine(Machine: int, quiet: bool = False) -> str
```

## Parameters

<ParamField path="Machine" type="int" required>
  id of machine to display
</ParamField>

<ParamField path="quiet" type="bool" default="False">
  only display numeric ids
</ParamField>

## Returns

`str` — Result from the API call.

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.show_machine(Machine=12345)
print(result)
```
