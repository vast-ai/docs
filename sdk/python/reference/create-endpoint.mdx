Create a new serverless endpoint group.

## Signature

```python
VastAI.create_endpoint(
    min_load: float = 0.0,
    target_util: float = 0.9,
    cold_mult: float = 2.5,
    cold_workers: int = 5,
    max_workers: int = 20,
    endpoint_name: Optional[str] = None
) -> str
```

## Parameters

<ParamField path="min_load" type="float" default="0.0">
  Minimum load (requests/sec) below which idle workers are scaled down.
</ParamField>

<ParamField path="target_util" type="float" default="0.9">
  Target worker utilization ratio (0.0–1.0); workers are added to stay at this level.
</ParamField>

<ParamField path="cold_mult" type="float" default="2.5">
  Multiplier applied to warm worker count for pre-warming cold workers.
</ParamField>

<ParamField path="cold_workers" type="int" default="5">
  Number of pre-warmed (cold) workers to keep ready.
</ParamField>

<ParamField path="max_workers" type="int" default="20">
  Maximum number of workers allowed in this endpoint.
</ParamField>

<ParamField path="endpoint_name" type="Optional[str]">
  Human-readable name for the endpoint.
</ParamField>

## Returns

`str` — Result from the API call.

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.create_endpoint()
print(result)
```
