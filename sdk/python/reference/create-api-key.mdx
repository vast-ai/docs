Create a new API key.

## Signature

```python
VastAI.create_api_key(
    name: Any,
    permissions: Any,
    key_params: Any = None
) -> dict
```

## Parameters

<ParamField path="name" type="Any" required>
  name of the api-key
</ParamField>

<ParamField path="permissions" type="Any" required>
  
</ParamField>

<ParamField path="key_params" type="Any">
  optional wildcard key params for advanced keys
</ParamField>

## Returns

`dict`

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.create_api_key(name="value", permissions="value")
print(result)
```
