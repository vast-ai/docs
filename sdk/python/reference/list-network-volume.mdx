List disk space for rent as a network volume.

## Signature

```python
VastAI.list_network_volume(
    disk_id: int,
    price_disk: Optional[float] = 0.15,
    end_date: Optional[str] = None,
    size: Optional[int] = 15
) -> dict
```

## Parameters

<ParamField path="disk_id" type="int" required>
  id of network disk to list
</ParamField>

<ParamField path="price_disk" type="Optional[float]" default="0.15">
  storage price in $/GB/month, default: $%(default).2f/GB/month
</ParamField>

<ParamField path="end_date" type="Optional[str]">
  contract offer expiration - the available until date (optional, in unix float timestamp or MM/DD/YYYY format), default 1 month
</ParamField>

<ParamField path="size" type="Optional[int]" default="15">
  size of disk space allocated to offer in GB, default %(default)s GB
</ParamField>

## Returns

`dict`

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.list_network_volume(disk_id=12345)
print(result)
```
