Synchronous Vast.ai API client for marketplace search and instance lifecycle. Returns typed result wrappers (`SyncOffer`, `SyncInstance`) that expose follow-up operations directly on the object.

For an async variant with the same surface, see [`AsyncClient`](/sdk/python/reference/async-client).

## Import

```python
from vastai import SyncClient
```

## Constructor

```python
SyncClient(
    api_key: object = <from env / file>,
    vast_server: str = "https://console.vast.ai",
)
```

<ParamField path="api_key" type="Optional[str]">
  Vast API key. Resolution order: explicit argument → `VAST_API_KEY` env var → `$XDG_CONFIG_HOME/vastai/vast_api_key` → `~/.vast_api_key` (legacy).
</ParamField>

<ParamField path="vast_server" type="str" default='"https://console.vast.ai"'>
  Vast API base URL.
</ParamField>

## Methods

### search

```python
search(
    query: Query,
    type_: Optional[str] = None,
    order: Optional[list] = [["score", "desc"]],
    limit: Optional[int] = None,
    allocated_storage: Optional[float] = None,
    target_reliability: Optional[float] = None,
    disable_bundling: Optional[bool] = None,
    external: Optional[bool] = None,
    extra_ids: Optional[list] = None,
    has_avx: Optional[int] = None,
    gpu_option: Optional[Any] = None,
    show_incompatible: Optional[Any] = None,
    sort_option: Optional[Any] = None,
    template_id: Optional[Any] = None,
) -> list[SyncOffer]
```

Search available offers. `query` is a `vastai.data.query.Query` built via the column DSL (`gpu_name == RTX_4090`, `gpu_ram >= 24`, etc.).

### create_instance

```python
create_instance(offer_id: int, config: InstanceConfig) -> SyncInstance
```

Accept an offer and create an instance. Returns a `SyncInstance` wrapper you can call lifecycle methods on.

### show_instances

```python
show_instances() -> list[SyncInstance]
```

Return all instances owned by the authenticated user.

### destroy_instance

```python
destroy_instance(instance_or_id: Union[SyncInstance, int]) -> None
```

Destroy a running or stopped instance. Accepts either the numeric ID or a `SyncInstance` wrapper returned from `search`/`create_instance`/`show_instances`.

## Example

```python
from vastai import SyncClient
from vastai.data.query import gpu_name, RTX_4090
from vastai.data.instance import InstanceConfig

client = SyncClient(api_key="YOUR_API_KEY")

offers = client.search(
    query=(gpu_name == RTX_4090),
    limit=5,
)

config = InstanceConfig(image="pytorch/pytorch:latest", disk=20)
instance = client.create_instance(offer_id=offers[0].id, config=config)
print(instance.id)
```
