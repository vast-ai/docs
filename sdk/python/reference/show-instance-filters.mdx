Return distinct filterable values for instances.

## Signature

```python
VastAI.show_instance_filters() -> list
```

## Returns

`list`

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.show_instance_filters()
print(result)
```
