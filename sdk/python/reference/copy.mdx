Copy files between instances.

## Signature

```python
VastAI.copy(
    src: str,
    dst: str
) -> dict
```

## Parameters

<ParamField path="src" type="str" required>
  Source location for copy operation (supports multiple formats)
</ParamField>

<ParamField path="dst" type="str" required>
  Target location for copy operation (supports multiple formats)
</ParamField>

## Returns

`dict`

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.copy(src="value", dst="value")
print(result)
```
