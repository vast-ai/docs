Update an SSH key.

## Signature

```python
VastAI.update_ssh_key(
    id: int,
    ssh_key: str
) -> dict
```

## Parameters

<ParamField path="id" type="int" required>
  id of the ssh key to update
</ParamField>

<ParamField path="ssh_key" type="str" required>
  new public key value
</ParamField>

## Returns

`dict`

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.update_ssh_key(id=12345, ssh_key="value")
print(result)
```
