Set a label on an instance.

## Signature

```python
VastAI.label_instance(
    id: int,
    label: str
) -> dict
```

## Parameters

<ParamField path="id" type="int" required>
  id of instance to label
</ParamField>

<ParamField path="label" type="str" required>
  label to set
</ParamField>

## Returns

`dict`

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.label_instance(id=12345, label="value")
print(result)
```
