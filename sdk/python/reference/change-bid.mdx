Change the bid price for an instance.

## Signature

```python
VastAI.change_bid(
    id: int,
    price: Optional[float] = None
) -> dict
```

## Parameters

<ParamField path="id" type="int" required>
  id of instance type to change bid
</ParamField>

<ParamField path="price" type="Optional[float]">
  per machine bid price in $/hour
</ParamField>

## Returns

`dict`

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.change_bid(id=12345)
print(result)
```
