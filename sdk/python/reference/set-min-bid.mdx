Set the minimum bid price for a machine.

## Signature

```python
VastAI.set_min_bid(
    id: int,
    price: Optional[float] = None
) -> dict
```

## Parameters

<ParamField path="id" type="int" required>
  id of machine to set min bid price for
</ParamField>

<ParamField path="price" type="Optional[float]">
  per gpu min bid price in $/hour
</ParamField>

## Returns

`dict`

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.set_min_bid(id=12345)
print(result)
```
