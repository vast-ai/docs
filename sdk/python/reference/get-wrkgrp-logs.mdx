Fetch logs for a worker group.

## Signature

```python
VastAI.get_wrkgrp_logs(
    id: int,
    level: int = 1,
    tail: Optional[int] = None
) -> dict
```

## Parameters

<ParamField path="id" type="int" required>
  id of workergroup to fetch logs from
</ParamField>

<ParamField path="level" type="int" default="1">
  log detail level (0 to 3)
</ParamField>

<ParamField path="tail" type="Optional[int]">
  
</ParamField>

## Returns

`dict`

## Example

```python
from vastai import VastAI

client = VastAI(api_key="YOUR_API_KEY")
result = client.get_wrkgrp_logs(id=12345)
print(result)
```
