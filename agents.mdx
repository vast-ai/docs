AI coding agents can search GPU offers, create and destroy instances, transfer files, and manage billing on Vast.ai — without you writing a single API call.

## Install the Skill

```bash
npx skills add vast-ai/vast-cli
```

This registers the Vast.ai skill with your agent. Supported: Claude Code, Cursor, Windsurf, Codex, and any agent that supports the [skills protocol](https://skills.sh).

## What's Included

Two skills ship in the same package:

| Skill | Use when |
|-------|----------|
| [CLI skill (`vastai/SKILL.md`)](https://raw.githubusercontent.com/vast-ai/vast-cli/master/vastai/SKILL.md) | Your agent runs shell commands — search offers, create instances, SSH, copy files, manage billing |
| [SDK skill (`vastai_sdk/SKILL.md`)](https://raw.githubusercontent.com/vast-ai/vast-cli/master/vastai_sdk/SKILL.md) | Your agent writes Python code using `from vastai import VastAI` or the async/sync clients |

The CLI skill includes non-obvious API quirks agents need to avoid silent failures (SSH key timing, VRAM units, instance response format, onstart size limits).

## Get Your API Key

Create an API key at [cloud.vast.ai/manage-keys](https://cloud.vast.ai/manage-keys/), then set it:

```bash
vastai set api-key YOUR_KEY
```

Or export it as an environment variable:

```bash
export VAST_API_KEY=YOUR_KEY
```

The CLI checks `$VAST_API_KEY` first, then `~/.config/vastai/vast_api_key`, then `~/.vast_api_key`.

## Example Prompts

Once the skill is installed, you can ask your agent directly:

- "Find the cheapest RTX 4090 with at least 2 GPUs and rent it with a PyTorch image"
- "Show my running instances and their costs"
- "Copy my local dataset to instance 12345 and run training"
- "Destroy all stopped instances"
- "Follow the [autoresearch example](/examples/ai-agents/autoresearch) and run it end to end"
