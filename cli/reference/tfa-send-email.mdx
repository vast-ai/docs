Request a 2FA Email verification code

## Usage

```bash
vastai tfa send-email
```

## Description

Request a two-factor authentication code to be sent via Email.

The secret token will be returned and must be used with 'vastai tfa activate'.

## Examples

```bash
vastai tfa send-email
```
