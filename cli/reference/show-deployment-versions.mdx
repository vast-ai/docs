Display versions for a deployment

## Usage

```bash
vastai show deployment-versions ID
```

## Arguments

<ParamField path="id" type="integer" required>
  id of deployment to show versions for
</ParamField>

## Description

Example: vastai show deployment-versions 1234

## Examples

```bash
vastai show deployment-versions <ID>
```
