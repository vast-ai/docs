Defragment machines

<Note>This is a **host** command, used for managing machines you are renting out on Vast.ai.</Note>

## Usage

```bash
vastai defragment machines IDs
```

## Arguments

<ParamField path="IDs" type="integer" required>
  ids of machines
</ParamField>

## Description

Defragment some of your machines. This will rearrange GPU assignments to try and make more multi-gpu offers available.
