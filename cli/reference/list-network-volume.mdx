[Host] list disk space for rent as a network volume

## Usage

```bash
vastai list network volume DISK_ID [options]
```

## Arguments

<ParamField path="disk_id" type="integer" required>
  id of network disk to list
</ParamField>

## Options

<ParamField path="--price_disk" type="number" default="0.15">
  storage price in $/GB/month, default: $%(default).2f/GB/month
</ParamField>

<ParamField path="--end_date" type="string">
  contract offer expiration - the available until date (optional, in unix float timestamp or MM/DD/YYYY format), default 1 month
</ParamField>

<ParamField path="--size" type="integer" default="15">
  size of disk space allocated to offer in GB, default %(default)s GB
</ParamField>
