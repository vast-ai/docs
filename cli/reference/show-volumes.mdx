Show stats on owned volumes.

## Usage

```bash
vastai show volumes [OPTIONS]
```

## Options

<ParamField path="-t" type="string" default="all">
  volume type to display. Default to all. Possible values are "local", "all", "network" (alias: `--type`)
</ParamField>

## Description

Show stats on owned volumes
