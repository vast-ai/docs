Display details of a single deployment

## Usage

```bash
vastai show deployment ID
```

## Arguments

<ParamField path="id" type="integer" required>
  id of deployment to show
</ParamField>

## Description

Example: vastai show deployment 1234

## Examples

```bash
vastai show deployment <ID>
```
