Show hosted machines

<Note>This is a **host** command, used for managing machines you are renting out on Vast.ai.</Note>

## Usage

```bash
vastai show machines [OPTIONS]
```

## Options

<ParamField path="-q" type="boolean">
  only display numeric ids (alias: `--quiet`)
</ParamField>

## Global Options

The following options are available for all commands:

| Option | Description |
| --- | --- |
| `--url URL` | Server REST API URL |
| `--retry N` | Retry limit |
| `--raw` | Output machine-readable JSON |
| `--explain` | Verbose explanation of API calls |
