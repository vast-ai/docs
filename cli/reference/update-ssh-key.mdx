Update an existing SSH key

## Usage

```bash
vastai update ssh-key ID SSH_KEY
```

## Arguments

<ParamField path="id" type="integer" required>
  id of the ssh key to update
</ParamField>

<ParamField path="ssh_key" type="string" required>
  new public key value
</ParamField>
