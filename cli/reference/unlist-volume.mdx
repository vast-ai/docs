unlist volume offer

<Note>This is a **host** command, used for managing machines you are renting out on Vast.ai.</Note>

## Usage

```bash
vastai unlist volume ID
```

## Arguments

<ParamField path="id" type="integer" required>
  volume ID you want to unlist
</ParamField>
