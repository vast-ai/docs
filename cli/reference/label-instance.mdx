Assign a string label to an instance

## Usage

```bash
vastai label instance <id> <label>
```

## Arguments

<ParamField path="id" type="integer" required>
  id of instance to label
</ParamField>

<ParamField path="label" type="string" required>
  label to set
</ParamField>
