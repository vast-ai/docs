Get the logs for an instance

## Usage

```bash
vastai logs INSTANCE_ID [OPTIONS]
```

## Arguments

<ParamField path="INSTANCE_ID" type="integer" required>
  id of instance
</ParamField>

## Options

<ParamField path="--tail" type="string">
  Number of lines to show from the end of the logs (default '1000')
</ParamField>

<ParamField path="--filter" type="string">
  Grep filter for log entries
</ParamField>

<ParamField path="--daemon-logs" type="boolean">
  Fetch daemon system logs instead of container logs
</ParamField>
