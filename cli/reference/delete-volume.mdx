Delete a volume

## Usage

```bash
vastai delete volume ID
```

## Arguments

<ParamField path="id" type="integer" required>
  id of volume contract
</ParamField>

## Description

Deletes volume with the given ID. All instances using the volume must be destroyed before the volume can be deleted.
